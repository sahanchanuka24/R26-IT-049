## 01. ABSTRACT

This project introduces an intelligent guidance and evaluation system for automobile repair students, utilizing real-time camera data and Augmented Reality (AR) technology. The system identifies engine components, analyzes student actions, and provides visual overlays to guide the repair process. It aims to bridge the gap between theoretical knowledge and practical application by providing real-time feedback and safety alerts.

**Keywords:** Augmented Reality, Machine Learning, Computer Vision, Automobile Repair Learning, Intelligent Evaluation.

---

## 02. INTRODUCTION

Practical training in automobile engineering often requires expert supervision to ensure accuracy and safety. This project develops an AR-based system that assists students in identifying engine parts and performing repairs correctly. Using vision-based recognition and AR overlays, the system provides a hands-on learning experience with immediate verification of tasks.

---

## 03. PROJECT COMPONENTS & METHODOLOGY

### 3.1 Vision-Based Engine Component Identification

- **Process:** Detects and identifies engine parts (e.g., Spark plug, Air filter, Battery) from a real-time camera feed.
- **Technology:** Machine Learning (ML), Computer Vision (CV), Deep Learning (CNN Models).

### 3.2 Student Task Recognition

- **Process:** Analyzes student interaction logs and movements to predict the ongoing task (e.g., "Inspecting Filter").
- **Technology:** AI Rule-Based System, Pattern Recognition, Sequence Modeling.

### 3.3 AR-Supported Task Verification

- **Process:** Compares the student's real-time actions against reference data. It provides AR arrows and labels to guide the user through the correct steps.
- **Technology:** AR Core / ARKit, Visual Similarity Matching.

### 3.4 Practical Evaluation & Safety Feedback

- **Process:** Automatically scores the student based on accuracy and time taken. It also generates critical safety alerts based on the specific component being handled.
- **Technology:** Scoring System, Safety Rule Reasoning Engine, ML-based Performance Analysis.
